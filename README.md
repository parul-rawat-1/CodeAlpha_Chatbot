This project is a Basic Rule-Based Chatbot developed in Python with a graphical user interface using Tkinter. The chatbot interacts with users through predefined 
responses and provides a simple chat experience.

Goal:
Build a simple rule-based chatbot that can interact with users through predefined responses.

Scope:
Accept user inputs such as:
"hello"
"how are you"
"what is your name"
"date"
"time"
"bye"

Provide predefined responses like:
"Hi!"
"I'm fine, thanks!"
"I am your Python Chatbot"
"Goodbye!"

Key Concepts Used:
`if-elif` conditional statements  
Functions  
Event loops  
Input/Output handling  
Basic GUI development using Tkinter

Features
Interactive chat interface
Graphical User Interface (GUI) built with Tkinter
Dynamic responses for greetings and common queries
Displays current date and time
Send button and Enter key support
Clear chat option
Scrollable chat window

Technologies Used
Python
Tkinter (GUI Library)
Random module
Datetime module
